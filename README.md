# React Quiz App 🎉

A simple and interactive Quiz App built using React.js. Test your knowledge and have fun!

## 🌟 Features

- Dynamic quiz questions and answers.
- Timer functionality for each question.
- Score calculation at the end of the quiz.

## 📚 Technologies Used

- React.js
- JavaScript (ES6+)
- CSS
